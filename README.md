This repo is a **signpost**. It holds no code and receives no updates. Everything —
issues, pull requests, releases, and history — lives in the
[pounce-ai](https://github.com/orgs/pounce-ai) org.

### If you have a local clone

```sh
git remote set-url origin https://github.com/pounce-ai/pounce.git
```

### If you're running the desktop Bridge

Nothing to do — 1.0.2 through 1.0.12 auto-update through this repo's
`/pounce/appcast.xml` feed, which is still served here and points at the new org's
releases. The next update migrates you to the new feed automatically.

<sub>It also serves the `privacy.html`, `index.html`, and `how-it-works.html` paths
that are baked into the published App Store listing and the shipped iOS app — GitHub
does not redirect `*.github.io` hosts, so these must stay reachable here.</sub>
